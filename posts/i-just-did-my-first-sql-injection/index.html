I just did my first SQL injection!

February 9, 2019 One minute read
hacking • ctf • security
hacksplaining • sql injection

… and it felt good!

This week I spent a couple of hours finding my way through Norwegian Police Security Services new job application which required “hacking” to apply. See the solution on Reddit, GitHub and Kode24. I had a lot of fun trying to figure out the puzzles and it got me excited about learning more.

While I liked the challenge, I do tend to like to learn in a more structured way. Enter Hacksplaining! I’ve allways known about SQL injection, but never actually taken the time to dig into the details behind it. Hacksplaining lesson on SQL injection does a really good job at showing how easy it can be and how to protect oneself against it.

That’s it, I’m officially hooked! I think CTF is gonna be a hobby of mine going forward.
